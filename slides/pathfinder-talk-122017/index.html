What is Pathfinder?

  • A project with two main functions:
    • A GPU rasterizer for fonts.
    • A path rendering component designed to allow WebRender to render SVG and canvas.
  • Written in Rust and GLSL (OpenGL ES 3), with a demo and testing framework written in TypeScript.

Pathfinder compared

A taxonomy of GPU vector graphics approaches:
  • Texture-based
    • One feature per texel (SDF)
    • k features per texel (mSDF)
    • Unbounded features per texel (Slug, Dobbie)
  • Geometry-based
    • Brute-force (Pathfinder 2 ECAA, NVPR)
    • Winding prepass (Pathfinder 2 MCAA/SSAA)
    • Winding postpass (Pathfinder 1)

Rendering modes

Pathfinder 2 is a hybrid renderer, offering the choice of different algorithms for different situations.
  • SSAA—Supersampling
    • Build a resolution-independent mesh; render with quadratic Loop-Blinn.
  • MCAA—Mesh Coverage Antialiasing
    • Render the mesh with analytic AA.
  • ECAA—Edge Coverage Antialiasing
    • Render directly from curves using brute force analytic AA.

Font features

Pathfinder's output is designed to closely resemble the native OS font appearance.
  • Subpixel antialiasing with color defringing
  • Gamma correction
  • Stem darkening
  • Ultra-slight (macOS-style) hinting
  • Faux bold
